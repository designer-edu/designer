Designer Objects and Grouping Objects¶

arc(color, start_angle, stop_angle, thickness, *args)¶

Function to make an arc.

Parameters

  • color (str or List[str]) – color to draw arc

  • start_angle (int) – angle to start drawing arc at

  • stop_angle (int) – angle to stop drawing arc at

  • thickness (int) – thickness of arc in pixels

  • args (two Tuples (left, top), (width, height) or four ints left, top, width, height) – left top corner of arc and width and height of arc

Returns

Arc object created

circle(color, radius, *args)¶

Function to create a circle.

Parameters

  • color – color of circle

  • radius – int, radius of circle in pixels

  • args – center of circle in x, y either as separate ints or as a tuple of ints

Returns

Circle object created

ellipse(color, args)¶

Function to make an ellipse.

Parameters

  • color (str or List[str]) – color of ellipse

  • args (two Tuples (left, top), (width, height) or four ints left, top, width, height) – left top corner of ellipse and width and height of ellipse

Returns

Ellipse object created

group(*objects)¶

Function to group multiple objects together.

Parameters

objects (at least one DesignerObject) – collection of objects to be grouped together for collective functionality

image(path, *args)¶

Function to create an image.

Parameters

Returns

Image object

line(color, thickness, *args)¶

Function to create a line.

Parameters

  • thickness (int) – thickness of line in pixels

  • color (str or List[str]) – color of line

  • args (two tuples (start x, start y), (end x, end y) or four ints start x, start y, end x, end y) – start and end coordinates of line

Returns
